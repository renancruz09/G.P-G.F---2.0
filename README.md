# G.P-G.F---2.0
O Sistema de Gestão de Vendas automatiza o controle comercial de pequenos negócios via SQLite. Substitui planilhas manuais, garantindo segurança e dados íntegros. Com arquitetura CRUD, possui o Cadastro (Create), Edição (Update) e Exclusão (Delete) concluídos. A Listagem (Read) e o Menu Principal estão em fase de planejamento para fechar o app.
